# Security Policy

## Supported Versions

I'm trying to actively provide security updates for the current stable release. When a new major or minor version is released, support for the previous version typically ends.

| Version | Supported          |
| ------- | ------------------ |
| v1.1.x   | :white_check_mark: |
| v1.0.x   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.** I take the security of Resonance seriously. If you believe you have found a security vulnerability, please help us fix it by reporting it privately.
## How to Report

You can report vulnerabilities in **GitHub Private Reporting** (Preferred): Go to the Security tab of this repository and select "Report a vulnerability". This keeps the conversation private and integrated with the codebase.
